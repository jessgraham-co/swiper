1
Open Instagram in Safari
Go to instagram.com in Safari and make sure you're logged in to your account. Keep that tab open — you'll need it.
2
Create the Shortcut (no pasting!)
Open Shortcuts → tap + → name it Swiper.

Add Action 1: search "Get Contents of URL" → tap it → tap the URL field and type your GitHub Pages URL for swiper-shortcut.js (e.g. https://yourusername.github.io/swiper/swiper-shortcut.js).

Add Action 2: search "Run JavaScript on Webpage" → tap it → tap the script field → tap Variable → choose Contents of URL from Action 1.

Tap Done. That's it — no script pasting ever needed.
The Shortcut fetches the latest script live each time. If the script ever updates, you get it automatically.
3
Add to Share Sheet & run
Go to instagram.com in Safari (logged in). Tap Share → Edit Actions → find Swiper → tap + → Done.

Now tap Share → tap Swiper. The UI launches on top of Instagram.
Instagram rate-limits unfollows. This app enforces a 1-2 second delay and auto-pauses every 50 unfollows. Stay under ~150/hour.
